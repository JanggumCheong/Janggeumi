# images — 장금이 이미지 작업 스킬

장금이 프로젝트에서 사용하는 이미지 자산을 일관되게 생성하고 변환한다.

## 먼저 확인할 것

작업 전에 필요한 기준 문서를 읽는다.

- 식자재 이미지 생성: [ingredient-image.md](ingredient-image.md)
- 장금이 캐릭터 이미지 생성: [janggeumi-character.md](janggeumi-character.md)
- PNG to WebP 변환: [png-to-webp.md](png-to-webp.md)
- 화면 톤 확인이 필요하면 `../design/design-system.md`를 참고한다.

## 공통 원칙

- 기존 `.codex/skills/design`과 `.codex/skills/product`의 방향을 해치지 않는다.
- 이미지는 전통 장식보다 현대적인 서비스 자산으로 보이게 만든다.
- 식자재는 실제로 알아볼 수 있어야 하고, 장금이는 보조 안내자 역할을 넘지 않는다.
- 결과 파일명은 영문 소문자와 하이픈을 사용한다.
- 원본 PNG가 필요한 작업이면 원본을 보존하고 WebP를 별도 파일로 만든다.

## 작업 분기

- "식자재 이미지 만들어줘", "재료 카드 이미지", "수박 이미지" 같은 요청이면 `ingredient-image.md`를 따른다.
- "장금이 이미지", "캐릭터 표정", "마스코트" 같은 요청이면 `janggeumi-character.md`를 따른다.
- "png를 webp로", "이미지 최적화", "webp 변환" 같은 요청이면 `png-to-webp.md`를 따른다.

## 마무리

- 생성/변환한 파일 경로를 사용자에게 알린다.
- 사용한 기준, 비율, 저장 위치, 변환 품질 옵션을 짧게 남긴다.
- 기존 이미지나 다른 사람이 만든 파일을 덮어써야 하면 먼저 사용자에게 확인한다.
