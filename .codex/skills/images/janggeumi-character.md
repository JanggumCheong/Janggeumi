# 장금이 캐릭터 이미지 생성 기준

장금이는 서비스를 설명하는 보조 안내자다. 화면의 주인공은 사용자의 식자재 문제이고, 장금이는 친근하게 판단을 돕는다.

## 기준 이미지

캐릭터 톤이 필요하면 다음 파일을 먼저 참고한다.

```text
.codex/skills/design/Janggeumi-character.png
```

기존 기준 이미지를 수정하지 않는다. 새 이미지는 별도 파일로 만든다.

## 사용 상황

- 메인 페이지 안내 영역
- 빈 상태 또는 도움말
- 추천, 주의, 완료, 축하 메시지
- 장금이 한 줄 코멘트 옆 보조 이미지

## 캐릭터 방향

- 친근하고 똑똑한 살림 고수 느낌을 유지한다.
- 2D 일러스트 스타일을 기본으로 한다.
- 2~3등신의 작고 귀여운 비율을 유지한다.
- 전통 콘셉트가 화면 전체를 지배하지 않게 한다.
- 한복 요소가 있더라도 현대적인 앱 일러스트처럼 보이게 한다.
- 표정은 따뜻하고 명확하게 한다.
- 과장된 포즈보다 작고 또렷한 안내자 느낌을 우선한다.

## 복장과 헤어 시그니처

장금이 캐릭터는 다음 복장과 헤어 요소를 일관되게 유지한다.

- 붉은색 머리수건 또는 머리띠를 착용한다.
- 검은색 또는 짙은 갈색 머리는 가운데 가르마를 기준으로 단정하게 정리한다.
- 머리 뒤쪽에는 낮게 묶은 머리나 땋은 머리 느낌을 둔다.
- 상의는 연한 민트/세이지 그린 계열 저고리 느낌을 유지한다.
- 깃, 소매 끝, 고름에는 짙은 자주색 또는 와인색 포인트를 둔다.
- 하의는 짙은 남색 치마와 밝은 아이보리 앞치마 조합을 기본으로 한다.
- 전체 복장은 전통 한복을 그대로 재현하기보다, 현대적인 2D 앱 마스코트에 맞게 단순화한다.

참고 이미지는 복장, 머리수건, 색 조합, 전체 인상을 맞추기 위한 기준이다. 동일한 포즈, 동일한 표정, 동일한 장식 디테일, 동일한 구도는 복제하지 않는다.

## 상황별 표정

- 안내: 차분한 미소, 손짓, 설명하는 자세
- 추천: 자신감 있는 미소, 작은 포인트 제스처
- 주의: 부드러운 경고 표정, 놀라게 하지 않는 손짓
- 완료: 밝은 미소, 가벼운 축하 제스처
- 빈 상태: 기다리는 표정, 부담 없는 안내

## 프롬프트에 포함할 요소

- 장금이 캐릭터
- 2D app mascot illustration
- 2 to 3 head-tall chibi proportion
- red headscarf
- center-parted dark hair with a low tied or braided shape
- pale mint sage jeogori-style top
- burgundy collar, sleeve, and ribbon accents
- dark navy skirt and ivory apron
- 상황과 감정
- 사용할 화면 위치
- modern friendly app mascot
- clean background
- soft colors
- consistent character design
- use the reference image only as an identity and outfit guide, not as an exact copy

## 피해야 할 요소

- 사극 포스터 같은 과한 연출
- 붓글씨, 두루마리, 궁중 배경, 단청 장식
- 화면을 압도하는 큰 캐릭터
- 지나치게 유아적인 표정
- 기존 기준 이미지와 다른 얼굴, 머리, 복장 인상

## 파일명 규칙

```text
janggeumi-<situation>.webp
```

예시:

```text
janggeumi-guide.webp
janggeumi-warning.webp
janggeumi-complete.webp
```

## 확인 기준

- 기존 장금이 기준 이미지와 같은 캐릭터로 보이는가?
- 화면의 핵심 정보보다 튀지 않는가?
- 현대적이고 간결한 장금이 디자인 톤과 맞는가?
- 작은 크기에서도 표정과 역할이 읽히는가?
