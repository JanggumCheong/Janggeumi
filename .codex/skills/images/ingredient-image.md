# 식자재 이미지 생성 기준

식자재 이미지의 목표는 사용자가 재료를 즉시 알아보고, 신선도와 형태를 빠르게 판단하게 하는 것이다.

## 사용 상황

- 메인 페이지 추천 식자재 카드
- 식자재 목록 카드
- 식자재 상세 화면 히어로 또는 보조 이미지
- 구매 기준의 좋은 예시와 피할 예시
- 보관 방법 카드 또는 보관 상세 이미지

## 이미지 방향

- 실제 식자재처럼 보이는 사실적인 이미지로 만든다.
- 재료가 화면의 주인공이어야 한다.
- 배경은 밝고 깨끗하게 유지한다.
- 과한 소품, 전통 문양, 식탁 연출, 복잡한 배경은 피한다.
- 신선함, 형태, 색, 표면 질감이 잘 보이게 한다.

## 기본 구도

- 카드용: 정사각형 또는 4:3 비율.
- 히어로용: 16:9 또는 넓은 가로 비율.
- 배경은 `#FBFBFA` 또는 흰색 계열에 어울리게 한다.
- 식자재는 중앙 또는 약간 아래쪽에 배치한다.
- 그림자는 아주 약하게 둔다.

## 공통 프롬프트 규칙

### 프롬프트에 포함할 요소

- 재료명
- 사용 위치: 카드, 히어로, 구매 기준, 보관 방법 등
- 상태: 통째, 자른 단면, 손질 전, 손질 후
- 목적: 대표 이미지, 좋은 예시, 피할 예시, 보관 방법
- 보관 맥락: 실온, 냉장, 냉동, 밀폐, 소분, 키친타월 사용 등
- 배경: clean bright background, minimal props
- 조명: soft natural light
- 스타일: realistic food photography, modern app asset

### 피해야 할 요소

- 만화풍 또는 과도한 일러스트
- 전통 한지, 붓글씨, 궁중 소품
- 어두운 배경
- 재료를 알아보기 어려운 과한 클로즈업
- 실제 재료와 다른 색감 또는 형태
- 사람이 들고 있거나 조리 장면이 주인공이 되는 이미지

### 파일명 규칙

```text
<ingredient-slug>-<usage>.webp
```

예시:

```text
watermelon-card.webp
watermelon-good-example.webp
watermelon-bad-example.webp
green-onion-storage.webp
potato-room-storage.webp
```

### 확인 기준

- 재료가 1초 안에 식별되는가?
- 카드 크기로 줄여도 형태가 유지되는가?
- 배경과 소품이 재료보다 튀지 않는가?
- 장금이 디자인 시스템의 현대적이고 간결한 톤과 맞는가?

## 이미지 유형별 기준

### 기본 메인 이미지

메인 페이지, 식자재 목록, 추천 카드에 쓰는 대표 이미지다.

- 식자재 하나의 대표 인상을 명확하게 보여준다.
- 손질, 조리, 보관 상황보다 재료 자체가 주인공이어야 한다.
- 통째 모습이나 가장 익숙한 형태를 우선한다.
- 여러 화면에서 재사용될 수 있도록 배경과 구도를 단순하게 유지한다.

### 추천 식자재 배너 이미지

추천 식자재 배너 이미지는 메인 화면에서 오늘의 추천 재료 또는 계절 식자재를 소개할 때 사용하는 가로형 이미지다. 사용자가 한눈에 추천 재료를 알아볼 수 있도록 식자재 자체를 가장 명확하게 보여준다.

#### 추천 식자재 배너 기본 방향

- 추천 식자재가 이미지의 유일한 주인공이어야 한다.
- 장금이 캐릭터는 추천 식자재 배너 이미지에 넣지 않는다.
- 이미지 안에 텍스트, 라벨, 스티커, 말풍선은 넣지 않는다.
- 문구는 UI 레이어에서 올릴 수 있도록 이미지 한쪽에 깨끗한 여백을 확보한다.
- 전체 분위기는 밝고 깨끗하며, 식자재 관리 앱에 어울리는 신뢰감과 신선함을 유지한다.
- 과한 광고 이미지처럼 보이지 않게 한다.

#### 추천 식자재 배너 구도

- 배너 비율은 기본적으로 16:9 가로형을 사용한다.
- 식자재는 화면의 한쪽, 보통 오른쪽 영역에 크게 배치한다.
- 반대쪽에는 UI 텍스트가 올라갈 수 있는 깨끗한 여백을 둔다.
- 식자재는 전체 형태가 잘리지 않게 배치한다.
- 모바일 화면에서도 1초 안에 재료를 식별할 수 있어야 한다.
- 원물과 단면을 함께 보여주면 재료의 신선함과 맛을 더 잘 전달할 수 있다.
- 단면을 사용할 경우 과하게 많지 않게 배치하고, 원물이 여전히 주인공으로 보여야 한다.

#### 추천 식자재 배너 식자재 표현

- 식자재는 실제 사진처럼 현실적인 질감과 색감을 유지한다.
- 제철 추천 재료는 신선함, 수분감, 색감, 표면 질감이 잘 드러나게 한다.
- 수박처럼 내부 색이 매력적인 재료는 원물과 단면을 함께 배치할 수 있다.
- 단면은 먹음직스럽게 보이되, 조리 장면처럼 보이면 안 된다.
- 소품은 최소화하고, 식자재보다 눈에 띄면 안 된다.

#### 추천 식자재 배너 배경과 색감

- 배경은 `#FBFBFA` 또는 밝은 크림/연두 계열을 사용한다.
- 배경은 단순하고 깨끗해야 한다.
- 필요하면 아주 은은한 연두 톤을 섞어 장금이 메인 컬러와 연결한다.
- 식자재 색이 가장 먼저 보이도록 배경 대비를 확보한다.
- 그림자와 하이라이트는 부드럽게 사용한다.
- 어두운 배경, 강한 그라데이션, 복잡한 패턴은 피한다.

#### 추천 식자재 배너 피해야 할 요소

- 장금이 캐릭터
- 이미지 안에 직접 들어간 텍스트
- 라벨, 스티커, 말풍선, 광고형 배지
- 식자재를 알아보기 어려운 과한 클로즈업
- 너무 많은 소품
- 어둡거나 탁한 색감
- 실제 식자재와 다른 형태나 색
- 조리 장면이 주인공이 되는 연출

#### 추천 식자재 배너 파일명 규칙

```text
<ingredient-slug>-recommend-banner.webp
```

예시:

```text
watermelon-recommend-banner.webp
apple-recommend-banner.webp
shine-muscat-recommend-banner.webp
```

### 구매 기준 이미지

식자재를 고를 때 좋은 예시와 피해야 할 예시를 설명하는 이미지다.

#### 구매 기준 공통 규칙

- 사용자가 차이를 눈으로 배울 수 있게 기준이 되는 부분을 명확히 보여준다.
- 좋은 예시는 신선함, 색, 모양, 표면 상태가 또렷하게 보이게 한다.
- 피할 예시는 무름, 변색, 상처, 마름, 과숙 등 문제 지점이 과장 없이 보이게 한다.
- 특정 기준을 보여줄 때는 재료 전체와 기준 부위가 함께 보이게 한다.
- 텍스트, 화살표, 라벨은 이미지 안에 넣지 않는다. 설명은 UI 텍스트가 담당한다.

#### 구매 기준 이미지 구성

- 이미지는 항상 하나의 장면(Scene)만 생성한다.
- 화면을 좌우 또는 상하로 나누어 비교하는 구성을 사용하지 않는다.
- 하나의 식자재만 중심으로 표현한다.
- 식자재와 필요한 소품은 화면 안에 모두 완전히 들어오도록 구성한다.
- 피사체가 프레임 밖으로 잘리거나 일부만 보이지 않게 한다.
- 식자재 전체가 자연스럽게 보이는 거리에서 촬영한 것처럼 표현한다.
- 배경은 반드시 흰색이어야 한다.
- 이미지는 WebP 파일로 생성한다.

#### 구매 기준 이미지 스타일

- 실제 카메라로 촬영한 것 같은 자연스러운 사진 스타일로 표현한다.
- 3D 렌더링, CG, 일러스트, 게임 그래픽 느낌은 사용하지 않는다.
- 과도한 선명도, 과포화된 색감, 인위적인 광택은 피한다.
- 부드러운 자연광 아래에서 촬영한 것처럼 표현한다.
- 현실적인 색감과 질감을 유지한다.

#### 구매 기준 사람 표현

- 사람 손이 필요한 기준은 손이 보조 역할만 하게 한다.
- 손, 팔, 사람 얼굴이 식자재보다 눈에 띄면 안 된다.

#### 구매 기준별 연출 예시

- 꼭지 확인: 식자재의 꼭지나 줄기 부분이 잘 보이게 한다.
- 모양 확인: 식자재의 전체 윤곽, 좌우 균형, 둥근 정도, 길쭉한 정도가 잘 보이게 한다.
- 줄무늬 선명도: 표면의 줄무늬, 색 대비, 패턴이 잘 보이게 한다.
- 색과 윤기: 껍질의 색, 광택, 생기 있는 표면이 잘 보이게 한다.
- 상처와 무름: 멍, 상처, 갈변, 눌린 부분이 과장 없이 보이게 한다.
- 단면 확인: 자른 단면의 색, 수분감, 조직감이 잘 보이게 한다.
- 두드렸을 때 소리: 사람 손이 식자재를 가볍게 두드리는 제스처를 보여준다.
- 무게감 확인: 사람의 손이 식자재를 들어 올리거나 받쳐 무게감을 확인하는 장면을 보여준다.
- 향 확인: 사람 손이나 얼굴을 과하게 넣지 않고, 재료를 가까이 확인하는 맥락만 최소한으로 보여준다.

#### 구매 기준 추가 규칙

- 모양 확인 이미지는 식자재 전체가 프레임 안에 들어오도록 한다.
- 여러 개를 비교하는 경우에는 좋은 예시와 피해야 할 예시를 같은 크기와 각도로 배치한다.

### 보관 방법 이미지

보관 방법 카드나 상세 화면에서 사용자가 따라 할 상태를 보여주는 이미지다.

- 재료의 보관 상태가 한눈에 보여야 한다.
- 냉장, 냉동, 실온, 손질 후, 소분 후처럼 보관 맥락을 명확히 드러낸다.
- 용기, 지퍼백, 키친타월, 랩 등 보관 도구는 필요한 경우에만 최소한으로 사용한다.
- 도구가 재료보다 더 눈에 띄지 않게 한다.
- 실제 주방 사진처럼 복잡하게 연출하지 않고 앱 자산처럼 정돈된 구도를 유지한다.
- 보관 방법이 잘못 전달될 수 있는 연출은 피한다.
- 배경은 반드시 흰색이어야 한다.
- 이미지 안에 텍스트는 절대 넣지 않는다.

#### 보관 방법별 연출 예시

- 통째로 실온 보관: 식자재 전체가 잘림 없이 보이게 한다. 하나의 큰 대표 이미지처럼 과일이나 채소의 전체 형태가 프레임 안에 들어와야 한다.
- 통째로 냉장 보관: 식자재 전체 형태가 보이되, 냉장 맥락은 과하게 연출하지 않는다. 필요한 경우 차가운 톤의 깨끗한 선반이나 용기만 최소한으로 사용한다.
- 잘라서 밀폐용기 보관: 잘린 식자재가 투명한 밀폐용기 안에 담겨 있는 상태를 보여준다. 자른 단면과 용기 뚜껑/밀폐감이 함께 읽혀야 한다.
- 밀폐용기 이미지는 기본적으로 뚜껑을 덮지 않는다. 재료가 잘 보이도록 열린 투명 용기 안에 담긴 상태를 보여주고, 밀폐감은 용기 형태나 가장자리로만 암시한다.
- 잘라서 랩으로 감싼 후 냉장: 잘린 단면이 랩으로 감싸인 느낌이 보이게 한다. 랩의 투명한 질감은 보이되 반사광이 과하지 않게 한다.
- 소분 보관: 여러 조각이 작은 용기나 지퍼백에 나뉘어 담긴 상태를 보여준다. 소분 단위가 명확해야 한다.
- 키친타월 보관: 키친타월이 재료를 받치거나 감싸 수분을 조절하는 맥락을 보여준다. 키친타월이 재료를 가리지 않게 한다.

보관 방법 이미지는 재료 상태와 보관 도구가 함께 읽혀야 한다. 다만 도구는 방법을 설명하기 위한 보조 요소이며, 재료보다 더 크게 보이면 안 된다.

### 처리 방법 이미지

처리 방법 이미지는 남은 식자재를 버리거나 손질하고, 다른 음식으로 활용하는 방법을 보여주는 이미지다.

콘텐츠 작성 주체에 따라 `UGC`와 `Guide`로 구분한다.

- `UGC`: 사용자가 직접 작성한 후기, 질문, 경험담, 생활 팁을 표현한다.
- `Guide`: 장금이가 정보를 정리하여 추천하는 처리 방법을 표현한다.
- 기본 비율은 가로 4:3으로 한다.
- 한 이미지에는 하나의 처리 행동이나 하나의 완성 결과만 보여준다.
- 식자재, 처리 행동 또는 완성 음식이 이미지의 주인공이어야 한다.
- 식자재의 실제 색, 형태, 껍질, 단면 구조를 정확하게 유지한다.
- 이미지 안에 텍스트, 라벨, 화살표, 스티커, 말풍선, 로고는 넣지 않는다.
- 화면을 나눈 비교 이미지나 여러 단계를 합친 콜라주는 만들지 않는다.

#### 처리 방법 UGC 타입

UGC 타입은 실제 사용자가 집에서 식자재를 처리하거나 활용하는 장면처럼 표현한다.

##### UGC 적용 기준

다음과 같은 콘텐츠는 UGC 타입으로 분류한다.

- 사용자의 실제 경험이나 후기를 담은 문장
- 다른 사용자에게 방법을 묻는 질문
- 구어체, 감탄, 축약어가 포함된 문장
- `ㅋㅋ`, `대박`, `국룰`, `해드셈`, `순삭`처럼 커뮤니티에서 사용하는 표현
- 사용자가 직접 발견한 생활 팁이나 레시피

##### UGC 이미지 방향

- 실제 가정집에서 촬영한 스마트폰 사진처럼 자연스럽게 표현한다.
- 사용자의 손이나 팔을 보조적으로 포함하여 직접 처리하는 느낌을 전달한다.
- 필요한 경우 몸통 일부까지 보여줄 수 있지만 얼굴은 기본적으로 제외한다.
- 사람보다 식자재와 처리 행동이 더 크게 보여야 한다.
- 한 이미지에는 한 사람만 등장하게 한다.
- 손, 팔, 피부색, 옷, 체형은 한 사람으로 일관되게 표현한다.
- 여러 사람의 손처럼 보이거나 성별과 신체 특징이 충돌하지 않게 한다.
- 손이 필요하지 않은 장면에는 억지로 사람을 넣지 않는다.
- 완벽하게 정돈된 광고 사진보다 실제 생활에서 볼 수 있는 자연스러운 구도를 사용한다.
- 손질된 식자재의 크기와 모양은 약간 불규칙하게 표현할 수 있다.
- 배경에는 일반 가정의 주방이나 식탁을 사용하되 지나치게 복잡하거나 지저분하게 만들지 않는다.
- 상하거나 버리는 식자재도 혐오스럽거나 과도하게 부패한 모습으로 표현하지 않는다.

##### 폐기 방법 UGC

폐기 방법 UGC는 사용자가 식자재를 직접 자르거나 분리하여 버리는 행동을 중심으로 보여준다.

- 자르기, 껍질 벗기기, 씨앗 분리하기, 꼭지 떼기, 줄기 분리하기 등의 행동을 보여준다.
- 가위, 칼, 도마, 음식물 쓰레기통, 종량제 봉투 등 필요한 도구만 사용한다.
- 어떤 부위를 분리하고 어디에 버리는지 한눈에 알아볼 수 있어야 한다.
- 칼이나 가위를 사용할 때는 손의 위치와 사용 자세를 안전하게 표현한다.
- 쓰레기통 내부를 지나치게 확대하거나 지저분하게 표현하지 않는다.
- 버리는 장면이 음식을 먹거나 조리하는 장면처럼 보이지 않게 한다.

##### 레시피 처리 UGC

레시피 처리 UGC는 사용자가 남은 식자재를 직접 요리로 활용하는 행동을 중심으로 보여준다.

- 썰기, 채 썰기, 으깨기, 섞기, 무치기, 끓이기 등의 조리 행동을 보여준다.
- 필요한 경우 조리 중인 음식과 완성된 음식 일부를 같은 장면 안에 배치할 수 있다.
- 실제 집밥처럼 자연스럽고 친근한 분위기를 유지한다.
- 곰팡이가 피거나 완전히 부패한 식자재를 요리에 사용하는 것처럼 표현하지 않는다.
- 먹을 수 있는 상태의 무른 재료, 갈변한 재료, 단맛이 떨어진 재료만 활용한다.
- 음식이 지나치게 완벽하거나 상업 광고 사진처럼 보이지 않게 한다.

#### 처리 방법 Guide 타입

Guide 타입은 장금이가 사용자에게 적절한 처리 방법을 추천하는 정돈된 안내 이미지로 표현한다.

##### Guide 적용 기준

다음과 같은 콘텐츠는 Guide 타입으로 분류한다.

- 식자재의 정확한 처리 방법을 설명하는 문장
- 음식물 쓰레기와 일반쓰레기 분류를 안내하는 문장
- 특정 부위를 분리하거나 잘게 잘라 배출하도록 권장하는 문장
- 남은 식자재를 활용하는 레시피를 정리한 문장
- `분류됩니다`, `배출해야 합니다`, `활용할 수 있습니다`, `권장합니다`와 같은 설명체 문장

##### Guide 이미지 방향

- 사람, 손, 팔, 얼굴을 넣지 않는다.
- 처리 과정이 아니라 처리 방법이 적용된 결과를 중심으로 보여준다.
- 밝고 깨끗한 배경과 부드러운 자연광을 사용한다.
- 현대적이고 간결한 장금이 앱의 이미지 톤을 유지한다.
- 장금이가 추천하는 이미지라는 의미이며 장금이 캐릭터를 직접 넣지 않는다.
- 궁중 소품, 전통 문양, 한지, 붓글씨 등의 연출을 사용하지 않는다.
- 소품은 처리 방법을 설명하는 데 필요한 것만 최소한으로 사용한다.
- 식자재와 필요한 도구가 서로 겹치지 않게 명확하게 배치한다.
- 안내 문구가 UI에서 올라갈 수 있도록 필요한 경우 깨끗한 여백을 확보한다.
- 이미지 안에 설명문, 라벨, 화살표, 아이콘, 배지를 직접 넣지 않는다.

##### 폐기 방법 Guide

폐기 방법 Guide는 식자재가 올바르게 분리되고 배출 준비가 완료된 상태를 보여준다.

- 음식물 쓰레기로 버리는 부위와 일반쓰레기로 버리는 부위를 명확하게 분리한다.
- 씨앗, 줄기, 꼭지, 껍질, 과육이 서로 섞여 보이지 않게 한다.
- 잘게 잘라야 하는 식자재는 알맞은 크기로 손질된 결과를 보여준다.
- 배출 용기가 필요한 경우 식자재보다 눈에 띄지 않게 보조적으로 배치한다.
- 올바른 배출 대상과 용기의 관계가 구도만으로 이해되어야 한다.
- 쓰레기통 내부를 지저분하거나 혐오스럽게 표현하지 않는다.
- 여러 처리 단계를 한 장에 나열하지 않는다.

##### 레시피 처리 Guide

레시피 처리 Guide는 장금이가 추천하는 완성 음식을 중심으로 보여준다.

- 완성된 음식 하나를 이미지의 유일한 주인공으로 표현한다.
- 음식의 형태, 색, 수분감, 바삭함이나 부드러움 등의 질감이 잘 보이게 한다.
- 조리 과정을 별도로 요청하지 않았다면 손질 전 식자재를 함께 나열하지 않는다.
- 고추장, 식초, 설탕, 소금 등의 양념 그릇을 별도로 배치하지 않는다.
- 도마, 칼, 믹싱 볼 등의 조리 도구를 기본적으로 넣지 않는다.
- 음식임을 알아보는 데 필요하지 않은 원물이나 껍질을 배경 소품으로 넣지 않는다.
- 그릇 하나와 최소한의 식기 또는 천 정도만 보조 소품으로 사용할 수 있다.
- 완성 음식이 일반적인 다른 요리로 오인되지 않도록 식자재의 형태적 특징을 유지한다.
- 상하거나 곰팡이가 핀 재료를 요리한 것처럼 표현하지 않는다.
