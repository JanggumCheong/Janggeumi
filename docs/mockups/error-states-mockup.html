확정 사항

일러스트로 응집 — 세 화면 모두 장금이 일러스트. 404="상품이 없어요"(가리킴), 에러="?" 갸웃(캐릭터 시트에서 크롭한 janggeumi-error.png). 같은 캐릭터라 톤 통일, 포즈로 상황 구분.
제목은 "잠시 문제가 생겼어요"로 통일, 보조 문구만 구분(상세=재료 정보 / 전역=화면). 액션 = 다시 시도(primary) + 홈으로(secondary). 404는 홈 CTA만(재시도 개념 없음).
정적→동적 충돌로 인한 500 버그 수정 완료 ([slug] force-dynamic). apple → 404 정상.
에러 계층은 상세 + 전역 2계층 (global-error 제외 — 앱 셸 유지가 마케팅상 유리).
코드 응집: 공통 ErrorState 컴포넌트로 상세·전역이 같은 레이아웃 공유(일러스트+제목+보조+액션).
